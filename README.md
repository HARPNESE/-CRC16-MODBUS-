一款强大简洁的自定义中控MODBUS代码计算器
软件特色：
A、支持自定义公式计算如10进制输入框A0101输入42，16进制输入框B0103可以直接输入A0101*3得到7E。同时支持跨计算行变量引用，例如B0202也可以输出A0101*3得到7E，方便在只有地址码为唯一变量时计算大量代码。
B、支持配置导入导出，在配置完每一行的计算公式后，可以直接导出配置文件，后续需要使用时直接导入即可，配置文件为.json格式。
C、支持自定义输入框和计算项数量设置，16进制输入框数量范围为1-20（一个输入框对应2字节16进制数，如A0），10进制为0-4，计算项1-30。匹配市面上绝大部分中控代码长度。
